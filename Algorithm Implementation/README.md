1. Breadth-First Search (BFS)
   
How it works: Explores nodes level by level from the start node using a queue (FIFO).

Applications:

Finding shortest path in unweighted graphs

Social networking (degree of connection)

Complexity:

Time: O(V + E)
Space: O(V)
